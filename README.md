NursesPrep
NursesPrep est une application web responsive dédiée aux étudiants en soins infirmiers. Elle centralise tous les outils nécessaires à la préparation des examens et au suivi de la formation.
Fonctionnalités

Quiz — Des séries de questions pour s'entraîner et tester ses connaissances
Flashcards — Des cartes mémoire pour réviser efficacement
Exercices & Cours — Du contenu pédagogique structuré par thématiques
Fiches de révision — Des fiches synthétiques pour aller à l'essentiel
Annales — Accès aux sujets des années précédentes
Bibliothèque de médicaments — Une base de données médicamenteuse intégrée
Suivi de progression — Tableau de bord personnalisé pour visualiser ses avancées
Système d'abonnement — Gestion des accès via des plans d'abonnement (intégration Stripe)
Profil utilisateur — Espace personnel configurable
Groupes & communauté — Espaces d'échange entre étudiants
Support — Système de tickets pour contacter l'équipe NursesPrep
Interface d'administration — Panneau admin permettant d'ajouter du contenu (quiz, cours, fiches, médicaments, annales), de gérer les abonnements et de répondre aux tickets

Stack technique
Frontend

React 19 — Interface utilisateur
React Router v7 — Navigation et gestion des routes
Tailwind CSS — Stylisation et design responsive
Framer Motion — Animations et transitions
Axios — Appels API vers le backend
Vercel Analytics — Suivi des performances

Backend

Node.js / Express 5 — Serveur et API REST
MongoDB / Mongoose — Base de données
JWT (jsonwebtoken) — Authentification et gestion des sessions
bcryptjs — Hashage des mots de passe
Stripe — Gestion des paiements et abonnements
Multer — Upload de fichiers
Nodemailer / Resend — Envoi d'emails
Web Push — Notifications push
PDF Parse — Extraction de contenu depuis des PDFs

Déploiement

Frontend déployé sur Vercel
Application responsive, accessible sur desktop et mobile
